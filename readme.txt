插件名称：同步到QQ空间

说明：发表日志时可以选择是否拷备一份至QQ空间，基于寒川版Qzone日志插件制作，增加发布文章时自选同步与否、加密日志不会被同步。

  安装：
    1、上传插件至博客插件目录。
    2、后台激活插件，并设置。
    3、发表文章时勾选Qzone即同步到QQ空间。
    4、注意：加密日志不会被同步。

注意：

1、本插件原理很简单，利用QQ邮箱发表日志。因此，在使用本插件前请先确认以开通QQ邮箱、QQ空间及QQ邮箱中IMAP服务功能。
